Antes de empezar

En este primer tutorial aprenderás a instalar GIT en el sitema operativo que tengas, ya sea Windows, Linux o Mac OS. Puedes desplazarte directamente a tu sistema operativo pulsando sobre el sistema operativo que tengas:

  1. Windows
  2. Linux
  3. Mac OS

Windows

  1. Descargar la última versión de GIT para WINDOWS.

  2. Iniciar el instalador, y seguir con todos los valores por defecto. Estos valores van bien para los usuarios nuevos.

  3. Abre una ventana de consola (cmd) y ejecuta los siguientes comandos para cambiar los datos de tu usuario y contraseña de GIT sustituyendo "defecto" por los tuyos. Estos datos estarán asociados a los commits que hagas:
    $ git config --global user.name "defecto"
    $ git config --global user.email "defecto@defecto.com"

  4. Finalmente, y de manera opcional, puedes instalar el ayudante de credenciales de GIT para Windows.

Linux

  1. Desde una terminal, ejecutaremos los siguientes comandos:
    sudo apt-get update
    sudo apt-get install libcurl4-gnutls-dev libexpat1-dev gettext libz-dev libssl-dev asciidoc xmlto docbook2x
  2. Clona el código fuente de GIT (o si no tienes ninguna versión instalada de GIT, descarga y extráela.
  3. Para poder construir GIT e instalarlo en /usr, hay que usar make:
    make all doc info prefix=/usr
    sudo make install install-doc install-html install-info install-man prefix=/usr

Mac OS

  1. Descargar la última versión de GIT
  2. Sigue el instalador como te indica.
  3. Abre la terminal y asegúrate de que la instalación se ha efectuado correctamente ejecutando git --version.
  4. Configura tu usuario y contraseña de GIT usando los siguientes comandos, sustituyendo "defecto" por los tuyos. Estos detalles estarán asociados con los commits que hagas:
    git config --global user.name "defecto"
    git config --global user.email "defecto@defecto.com"
